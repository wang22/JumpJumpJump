
# 命令说明
所有Key为onXxxxXxxx的属性为事件命令，Value为触发命令后发送给总调度的指令

## 事件
onCollisionByPlayer: 当与Player碰撞时产生的命令

## 指令
nextLevel: 进入到下一关
